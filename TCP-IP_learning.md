# TCP-IP详解卷1:协议 读书笔记

## 概述
 * 引言
     * TCP/IP协议族：允许运行不同操作系统的不同计算机互相通信【internet协议族】
 * 分层
     * 网络协议通常分层开发，每一层负责不同的通信功能，TCP/IP协议族是四个层次多个协议的组合
     * 链路层
     * 网络层
     * 运输层
     * 应用层
     ---
     * 互联网：网络的网络
         * 构造互联网：通过路由器、网桥连接两个以上的网络
             * 路由器：用于网络互联的硬件盒
             * 网桥：不同于路由器在网络层上连接，网桥在链路层进行互联
 * TCP/IP的分层
 * 互联网的地址
