## 记录笔记的目的是什么？

是为了更好的理解一件事，是为了能不断回忆，温故而知新，是为了养成习惯，更好的约束自己，提升自己。

## 那么记录笔记的要点又是什么呢？

保持简洁明了。记录方式简洁（网页github、goodnote）；分类方式简洁（学习工作、生活）

其次，要不断浏览，修改和归档

## 那么多信息，那么多需要记录的东西，要如何记录和储存呢？

记录：GitHub.com + goodnote

储存：GitHub + iCloud （从印象笔记迁移）
